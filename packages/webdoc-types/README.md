# @webdoc/types

This package contains all of the type definitions for docs (documentation blocks) and tags (@tag
decorators in the documentation).

## Overview

The are the core @webdoc/types Docs:

* BaseDoc
* ClassDoc
* FunctionDoc
* MethodDoc
* EventDoc
* TypedefDoc
* PropertyDoc

These the core @webdoc/types Tags:

* AccessTag
* ParamTag
* MemberTag
